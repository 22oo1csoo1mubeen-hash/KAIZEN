Dashboard Showcase Section

Immediately below the Hero Section.

This is the most important section of the landing page.

The dashboard should be presented as the heart of Kaizen.

The purpose of this section is to demonstrate how Kaizen brings together Habits, Goals and Expenses into a single unified experience.

This section should feel like a realistic product interface rather than a marketing card.

---

Dashboard Overview

Display a premium dashboard mockup that visually combines data from all three product pillars.

The dashboard should include:

* Daily Habit Progress
* Active Goals
* Monthly Expense Summary
* Overall Progress Indicator

The layout should immediately communicate:

"Everything important is visible in one place."

Life Score Widget

The dashboard should contain a central score representing overall progress.

Examples:

Life Score: 82

The score should be derived from:

* Habit Consistency
* Goal Progress
* Expense Discipline

This score acts as the primary visual highlight of the dashboard.

---

Visual Design

The dashboard should look like a real application.

Avoid:

* Fake charts
* Unrealistic statistics
* Marketing-style cards

Focus on:

* Product realism
* Clean layouts
* Professional spacing
* Premium interactions

The user should feel:

"I want to use this dashboard every day."

---

Animations

Dashboard elements should animate subtly:

* Counters increasing
* Progress rings filling
* Cards appearing smoothly
* Hover interactions

Animations should feel natural and premium.

No flashy effects.

---

Purpose

The dashboard should clearly demonstrate the value of Kaizen:

Track Habits.
Manage Goals.
Control Expenses.
Measure Growth.

All from one place.
